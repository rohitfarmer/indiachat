XMPP

Quick Start

  1. Install Blabbler.im (Android client).
  2. Create an account on https://anonym.im.
  3. Login in Blabber.im with your anonym account.
  4. Join xmpp:india@chat.disroot.org.
  5. Happy chatting!

Clients

  • Desktop: Gajim, a fully featured corss-platform XMPP client.
  • Android: Blabber.im, A Jabber/XMPP client for Android smartphones optimised for a unique mobile experience.
  • Web: Converse.js on Disroot.

IRC

Quick Start

  1. Install Revolution IRC (Android client).
  2. Add a new server: server name = DALnet; server address = irc.dal.net; nicknames = "preferred name".
  3. Connect to the DLAnet server/network.
  4. Join #india channel.
  5. Happy chatting!
